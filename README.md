## KYC Application

Android application for a digital banking platform that lets a relationship manager browse customer accounts and their KYC status.

**Tech Stack**

* Language: Kotlin

* UI Framework: Jetpack Compose 

* Architecture Pattern: MVVM

* Asynchronous Flow: Coroutines & Kotlin StateFlow

* Dependency Injection: Dagger Hilt

* Image Loading: Coil
